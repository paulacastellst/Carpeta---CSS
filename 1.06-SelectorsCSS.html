Selectors en CSS

Els selectors en CSS són els elements que tenim per triar a quin element de l'HTML donarem estil. Hi ha diferents tipus de selectors en CSS

  1. SELECTORS SIMPLES

    2. Podem dividir els selectors simples en dos grups principals.

    1. SELECTORS ELEMENTALS

      1. SELECTOR UNIVERSAL

      2. SELECTORS D'ETIQUETA

    2. SELECTORS D'ATRIBUTS

      1. SELECTORS D'ATRIBUTS ID i CLASSE

      2. SELECTORS D'ALTRES ATRIBUTS

  2. SELECTORS COMPOSTOS

    3. Podem dividir els selectors compostos en quatre grups principals.

    1. SELECTORS AGRUPATS

    2. SELECTORS COMBINADORS

      1. SELECTORS DESCENDENTS

      2. SELECTORS DE GERMA(NS)

      3. SELECTORS DE FILLS DIRECTES

    3. PSEUDOCLASSES

    4. PSEUDOELEMENTS

Com podeu veure hi ha moltes categories. No és massa important que us les apreneu perquè les aprendreu fent-les servir. Els hem classificat d'aquesta manera perquè necessitem alguna forma de categoritzar-les. En les següents classes, explicarem els tipus de selectors més bàsics i com es fan servir.
